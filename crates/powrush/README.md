## Quick Start

### Single-Player Mode (Recommended for now)

```bash
cargo run -p powrush --bin powrush-singleplayer
```

- `run` → Run a mercy-gated simulation cycle
- `status` → See world & player state
- `add Elara` → Add a new player
- `joy` → Check collective joy
- `help` → Full command list
