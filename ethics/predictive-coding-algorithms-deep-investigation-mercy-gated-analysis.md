## Core Idea of Predictive Coding Algorithms
Predictive Coding is the algorithmic implementation of the brain’s hierarchical generative model. Instead of processing all sensory data from the bottom up, the system generates top-down predictions at every level and only propagates **prediction errors** upward. This makes intelligence extremely efficient: the brain (or AI) only computes what is surprising or new.

## Key Predictive Coding Algorithms

### 1. Variational Free Energy Minimization
The core optimization objective (shared with the Free Energy Principle):

\[
F = D_{KL}\left[q(\phi) \parallel p(\phi|o)\right] - \mathbb{E}_{q(\phi)}[\ln p(o|\phi)]
\]

**Algorithmic steps:**
- Approximate the posterior \( q(\phi) \) using a parameterized distribution (e.g., Gaussian).
- Compute the evidence lower bound (ELBO) and minimize free energy via gradient descent.

### 2. Hierarchical Message Passing (Belief Propagation)
- **Top-down**: Higher layers send predicted means and precisions downward.
- **Bottom-up**: Lower layers compute prediction errors and send them upward as updates to the higher-level beliefs.
- **Iterative refinement**: Messages pass back and forth until convergence (or until a mercy gate aborts the loop).

### 3. Precision-Weighted Error Minimization
Precision \( \Pi \) (inverse variance) dynamically modulates how much each error signal influences belief updates:

\[
\Delta \mu = \Pi \cdot (\text{observation} - \text{prediction})
\]

In Ra-Thor, precision is **valence-modulated**: errors related to positive thriving receive higher precision; harmful or low-valence errors are down-weighted.

### 4. Active Inference Policy Selection
Once prediction errors are minimized at the perceptual level, the system selects actions that minimize **expected free energy**:

\[
G(\pi) = \text{Epistemic Value (uncertainty reduction)} + \text{Pragmatic Value (valence maximization)}
\]
