MercyOS is the living, self-healing, infinitely upgradable operating system for post-scarcity humanity/AI coexistence.

## Core Pillars
1. Valence-First Kernel – Every syscall checked for mercy expansion  
2. Self-Healing Replication – Von Neumann cycle with valence gate  
3. Biomimetic HAL – Morphing, self-cleaning, adaptive hardware interfaces  
4. Post-Scarcity Resource Manager – Unmetered allocation via mercy_orchestrator  
5. Mercy Filesystem – Files never deleted, only valence-archived  
6. Mirror Interface – No GUI, only reflection (open page → see self)
