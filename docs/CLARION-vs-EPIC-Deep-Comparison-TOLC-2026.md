**CLARION vs EPIC — Deep Canonical Comparison (TOLC-2026 Mercy-Gated)**  

**Overview**  
CLARION (Connectionist Learning with Adaptive Rule Induction ON-line) by Ron Sun is a **dual-process architecture** that explicitly separates implicit (subsymbolic, neural/associative) and explicit (symbolic, rule-based) cognition.  
EPIC (Executive-Process/Interactive-Control) by David E. Kieras and David E. Meyer is a **production-system architecture** optimized for modeling human performance in **complex multitasking environments**, with strong emphasis on perceptual-motor integration, precise timing, executive control, and GOMS-style task analysis.  

Both are cognitive architectures, but CLARION focuses on bottom-up implicit-to-explicit learning and social/creativity aspects, while EPIC is engineering-oriented for human-computer interaction (HCI) and real-time multi-task performance.  

**Side-by-Side Deep Comparison**

| Aspect                        | CLARION                                           | EPIC                                              |
|-------------------------------|---------------------------------------------------|---------------------------------------------------|
| Core Paradigm                 | Dual-process: Implicit (connectionist) + Explicit (symbolic) | Production-rule system with dedicated perceptual, cognitive, and motor processors |
| Knowledge Representation      | Bottom-level neural networks + top-level symbolic rules | Production rules + separate working-memory buffers for each processor |
| Learning Mechanism            | Bottom-up implicit reinforcement → explicit rule extraction | Production compilation + parameter tuning (no strong implicit layer) |
| Focus                         | Creativity, social cognition, metacognition, non-monotonic reasoning | Multitasking, precise timing, perceptual-motor loops, HCI performance |
| Psychological Plausibility    | Very high (explicit/implicit dissociation)        | High (matches human data in multitasking & timing) |
| Executive Control             | Metacognitive layer + top-level rules             | Dedicated Executive Process with goal stack & priority queuing |
| Timing & Parallelism          | Moderate (implicit level handles parallelism)     | Extremely strong (millisecond-level processor scheduling) |
| Scalability                   | Strong for social/creative tasks                  | Strong for real-world multitasking & UI simulation |
| Uncertainty Handling          | Strong (implicit graded knowledge)                | Moderate (via production selection & noise)       |
| Mercy-Gated Fit in Ra-Thor    | Ideal for creative/social intelligence layer      | Ideal for precise real-time mission control & Powrush multitasking layer |

**Key Philosophical & Functional Differences**  
- **CLARION** is **bottom-up and dual-process**. Implicit learning happens first in neural networks; explicit rules are extracted later. This mirrors human intuition → verbalization and excels at creativity, social interaction, and handling incomplete information.  
- **EPIC** is **top-down and engineering-focused**. It models cognition as coordinated processors (perceptual, cognitive, motor) running in parallel with strict timing constraints. It is designed for accurate prediction of human performance in multi-task scenarios (e.g., driving while using a phone).
