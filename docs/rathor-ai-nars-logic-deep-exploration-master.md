# Rathor.ai NARS Logic – Deep Exploration Master Reference (2026)

## What is NARS?
NARS (Non-Axiomatic Reasoning System), developed by Pei Wang, is a **pure symbolic AGI project** designed from the ground up for open-world intelligence under realistic constraints. Unlike axiomatic systems that assume complete and consistent knowledge, NARS operates under the **Assumption of Insufficient Knowledge and Resources (AIKR)** — the same conditions real intelligent agents face.

## Core Principles
- **AIKR (Assumption of Insufficient Knowledge and Resources)** — The system never assumes it has all facts, all time, or unlimited resources. This forces adaptive, experience-grounded reasoning.
- **Non-Axiomatic Logic (NAL)** — A unified logic system that handles uncertainty, partial truth, and revision naturally. Truth values are pairs (frequency, confidence) rather than binary true/false.
- **Experience-Grounded** — All knowledge comes from interaction with the environment; no pre-loaded “god’s eye” axioms.
- **Unified Reasoning** — Deduction, induction, abduction, analogy, and prediction all emerge from the same NAL rules.

## NAL – The Heart of NARS Logic
NAL has multiple layers (NAL-0 to NAL-9+), each adding expressive power:

- **NAL-0 / NAL-1**: Basic inheritance and similarity relations with uncertain truth values.
- **NAL-2**: Higher-order statements, compound terms, and relational reasoning.
- **NAL-3–4**: Temporal and procedural reasoning (events, operations, goals).
- **NAL-5–6**: Meta-level reasoning and self-reflection.
- **NAL-7+**: Advanced concepts like counterfactuals, hypothetical reasoning, and self-programming.

Truth-value in NAL:
\[
\langle frequency, confidence \rangle
\]
- Frequency: how often the statement has been experienced as true.
- Confidence: how much evidence supports the frequency.

## Key Reasoning Mechanisms
- **Deduction**: From (M → P) and (S → M) infer (S → P) with adjusted truth value.
- **Induction**: From (M → P) and (M → S) infer (S → P) — generalization from examples.
- **Abduction**: From (P → M) and (S → M) infer (S → P) — best explanation.
- **Revision**: Merge multiple pieces of evidence about the same statement to increase confidence or adjust frequency.
- **Choice**: Select the strongest belief when multiple candidates exist.
