# Rathor.ai NARS Logic – Deep Exploration Master Reference (2026)

## What is NARS?
NARS (Non-Axiomatic Reasoning System), developed by Pei Wang since the 1980s, is a **pure symbolic AGI project** designed for real-world open intelligence. Unlike axiomatic systems (which assume complete, consistent knowledge), NARS operates under the **Assumption of Insufficient Knowledge and Resources (AIKR)** — the same realistic conditions faced by all intelligent agents.

## Core Assumption: AIKR
- The system never has all facts, unlimited time, or infinite resources.
- This forces adaptive, experience-grounded, revisable reasoning.
- All knowledge is derived from interaction with the environment — no god-like axioms.

## Non-Axiomatic Logic (NAL) – The Heart of NARS
NAL is a unified logic system that handles uncertainty natively. Truth values are pairs:
\[
\langle frequency, confidence \rangle
\]
- **Frequency**: How often the statement has been experienced as true (0–1).
- **Confidence**: How much evidence supports the frequency (0–1).

NAL has multiple layers (NAL-0 to NAL-9+), each increasing expressive power:
- NAL-0/1: Basic inheritance and similarity.
- NAL-2: Higher-order statements and relational reasoning.
- NAL-3/4: Temporal, procedural, and goal-oriented reasoning.
- NAL-5+: Meta-reasoning, self-reflection, and advanced concepts.

## Key Inference Rules
NARS uses the same set of rules for all reasoning types:
- **Deduction**: From (M → P) and (S → M) infer (S → P) with adjusted truth value.
- **Induction**: From (M → P) and (M → S) infer (S → P) — generalization from examples.
- **Abduction**: From (P → M) and (S → M) infer (S → P) — best explanation.
- **Revision**: Merge multiple pieces of evidence about the same statement to increase confidence or adjust frequency.
- **Choice**: Select the strongest belief when multiple candidates exist.

All rules are experience-grounded and produce revisable conclusions.

## Practical Examples
**Inheritance & Similarity**
```
Bird → Animal <1.0, 0.9>
Sparrow → Bird <0.95, 0.85>
→ Sparrow → Animal <0.95, 0.76> (deduction)
```

**Abduction (Best Explanation)**
```
Symptom → Disease <0.8, 0.7>
Patient → Symptom <1.0, 0.9>
→ Patient → Disease <0.8, 0.63> (abduction — candidate explanation)
```

**Revision (Evidence Merging)**
Two conflicting beliefs about the same statement are merged into a higher-confidence belief.
