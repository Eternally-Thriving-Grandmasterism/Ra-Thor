## 1. Standard Attention (Quadratic Baseline)  
Given query \(Q\), key \(K\), value \(V\) matrices (\(n \times d\)):  
\[ \text{Attention}(Q,K,V) = \text{softmax}\left(\frac{QK^\top}{\sqrt{d}}\right)V \]  
Naïve implementation stores the full \(n \times n\) attention matrix → \(O(n^2)\) memory and bandwidth.

## 2. Tiling Strategy  
Split \(Q\), \(K\), \(V\) into blocks of size \(B_r \times d\) (query blocks) and \(B_c \times d\) (key/value blocks). Process one query block at a time against all key blocks, keeping only running statistics (no full matrix stored).

## 3. Online Softmax Derivation  
For a query block \(Q_i\) and successive key blocks \(K_j\):  
Compute partial scores \(S_{ij} = \frac{Q_i K_j^\top}{\sqrt{d}}\).  
Maintain running max \(m_i\) and sum \(l_i\):  
\[ m_i^{(j)} = \max(m_i^{(j-1)}, \max(S_{ij})) \]  
\[ l_i^{(j)} = l_i^{(j-1)} \cdot e^{m_i^{(j-1)} - m_i^{(j)}} + \sum_k e^{S_{ij}[k] - m_i^{(j)}} \]  
Final softmax probabilities for block \(j\):  
\[ P_{ij} = \frac{e^{S_{ij} - m_i^{(j)}}}{l_i^{(j)}} \]  

This avoids ever materializing the full \(n \times n\) matrix.

## 4. Forward Pass (Complete Algorithm)  
1. Load \(Q_i\) block.  
2. For each \(K_j, V_j\) block:

3. Final output: \(O_i = O_i / l_i\).
