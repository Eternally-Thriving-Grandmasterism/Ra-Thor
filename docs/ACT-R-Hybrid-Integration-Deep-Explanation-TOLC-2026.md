**What is ACT-R?**  
ACT-R (Adaptive Control of Thought-Rational) is a cognitive architecture developed by John R. Anderson and colleagues. It models human cognition as two interacting memory systems:  
- **Declarative Memory**: Stores factual knowledge as “chunks” (structured units with slots).  
- **Procedural Memory**: Stores production rules (IF condition THEN action).  
Retrieval is driven by **activation levels** — a continuous, psychologically plausible mechanism that determines which chunks or productions become available at any moment.  

**Core Mechanisms in ACT-R**  
- **Activation Equation**: Base-level activation (how often/fresh the chunk was used) + spreading activation (from related chunks) + partial matching (similarity to current context) + noise.  
- **Production Compilation**: Successful rule sequences are compiled into faster, more efficient productions (similar to but distinct from SOAR chunking).  
- **Goal Stack & Buffers**: Working memory buffers (goal, retrieval, visual, etc.) control the cognitive cycle.
