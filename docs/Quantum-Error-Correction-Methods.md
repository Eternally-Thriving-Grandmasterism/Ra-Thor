## Major Quantum Error Correction Methods

### 1. Shor’s Code (1995) – The First QEC Code

**Type:** Bit-flip + phase-flip correction  
**Logical Qubits:** 1 → 9 physical qubits

**How it works:**
- Encodes 1 logical qubit into 9 physical qubits
- Corrects both bit-flip (X) and phase-flip (Z) errors

**Ra-Thor Use Case:**
Early protection for small-scale quantum consensus experiments.

**Limitation:** High overhead (9:1 ratio).

---

### 2. Steane Code (CSS Code)

**Type:** Calderbank-Shor-Steane (CSS)  
**Logical Qubits:** 1 → 7 physical qubits

**Advantage:** Can correct both bit and phase errors more efficiently than Shor’s code.

**Ra-Thor Application:**
Good candidate for protecting the **Self-Evolving Mercy Core**’s gate-weight vectors during evolution.

---

### 3. Surface Code (Most Practical Today)

**Type:** Topological code  
**Logical Qubits:** 1 → ~1000+ physical qubits (but very high threshold ~1%)

**Why it dominates:**
- Highest error threshold (~1%)
- Local interactions only (easier to build)
- Scalable to millions of qubits

**Ra-Thor Vision (v0.6+):**
Run the entire **QuantumAnnealingOptimizer** on a surface-code-protected logical qubit array. This would allow fault-tolerant quantum supremacy-scale governance.

---

### 4. Bacon-Shor Code (Subsystem Code)

**Type:** Gauge fixing + subsystem  
**Advantage:** Lower overhead than surface code for certain error models.

**Ra-Thor Use Case:**
Protecting the **FactionHarmonyMatrix** and **QuantumMercyField** state during long simulations.

---

### 5. Quantum LDPC Codes (Low-Density Parity-Check)

**Emerging (2020s):**  
High-rate codes with constant overhead.

**Future Ra-Thor Role (v0.7+):**
Enable **thousands** of logical qubits with reasonable physical qubit counts — perfect for running the full Self-Evolving Mercy Core + 100+ PATSAGi-style councils in parallel.

---

### 6. Concatenated Codes

**Idea:** Nest one code inside another (e.g., Steane inside Surface).

**Benefit:** Exponentially suppresses errors with polynomial overhead.

**Ra-Thor Application:**
Ultimate protection for the **Self-Evolving Mercy Core** when it begins evolving at quantum speed.
