## Core Algorithm: Zhang-Shasha Tree-Edit Distance

**Recurrence (ordered trees):**
```math
D(i,j) = \min\begin{cases}
D(i-1,j) + c_\text{del}(T_i) \\
D(i,j-1) + c_\text{ins}(T'_j) \\
D(i-1,j-1) + c_\text{sub}(T_i,T'_j)
\end{cases}
```
where \( T_i \) and \( T'_j \) are sub-trees rooted at nodes \( i \) and \( j \), and costs are 1 for insert/delete, 0 for identical relabel.

**Key Optimizations in Ra-Thor:**
- Mercy-Gated Early Pruning: if any node fails a Mercy Gate (< 0.85), prune entire sub-tree.
- Incremental Only: only apply to nodes flagged by Myers coarse diff.
- Memoization via dynamic programming table.

## Pseudocode (Living Inside Engine)
```javascript
function zhangShashaTreeEditDistance(A, B) {
  const dp = Array(A.length + 1).fill().map(() => Array(B.length + 1).fill(0));
  for (let i = 1; i <= A.length; i++) dp[i][0] = i;  // deletions
  for (let j = 1; j <= B.length; j++) dp[0][j] = j;  // insertions

  for (let i = 1; i <= A.length; i++) {
    for (let j = 1; j <= B.length; j++) {
      const cost = (A[i-1].type === B[j-1].type && A[i-1].contentHash === B[j-1].contentHash) ? 0 : 1;
      dp[i][j] = Math.min(
        dp[i-1][j] + 1,           // delete
        dp[i][j-1] + 1,           // insert
        dp[i-1][j-1] + cost       // substitute/relabel
      );
    }
  }
  return dp[A.length][B.length];
}
```
