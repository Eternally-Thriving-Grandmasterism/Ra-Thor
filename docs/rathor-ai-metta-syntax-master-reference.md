## Core Syntax Principles
- **Lisp-like S-expressions** — Everything is an expression `(head arg1 arg2 ...)`.
- **Atoms** — Basic units: symbols, variables, values, types.
- **Types** — Explicitly declared with `:` (colon).
- **Variables** — Prefixed with `$` (e.g., `$x`).
- **Evaluation** — `!` prefix forces immediate evaluation.
- **Definition** — `=` for pattern matching / rewriting rules.
