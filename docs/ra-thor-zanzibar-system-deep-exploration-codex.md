- Stores **trillions** of relation tuples
- Handles **millions of authorization checks per second**
- p95 latency < 10 ms globally

- Provides strong **external consistency** via “zookies” (consistency tokens)
- Built on Spanner for storage, with heavy caching and sharding

- **Namespaces**: Define object types and their relations
- **Relation Tuples**: The atomic data unit

- **Expand API**: “Who has relation R on object O?”
- **Write API**: Add/remove tuples with causal consistency
