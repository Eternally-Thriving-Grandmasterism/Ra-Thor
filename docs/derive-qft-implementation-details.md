## 1. Definition of the Quantum Fourier Transform (QFT)
The QFT on n qubits transforms the computational basis state |x⟩ (x = 0 … 2ⁿ-1) as:
\[
\text{QFT}|x\rangle = \frac{1}{\sqrt{N}} \sum_{k=0}^{N-1} \exp\left( \frac{2\pi i x k}{N} \right) |k\rangle
\]
where \(N = 2^n\).

## 2. Circuit Construction
The QFT circuit uses Hadamard gates and controlled-phase gates \(R_k = \text{diag}(1, e^{2\pi i / 2^k})\):
- For each qubit j (0 ≤ j < n), apply H to qubit j.
- For each subsequent qubit k > j, apply controlled-\(R_{k-j+1}\) from qubit j to qubit k.
- Finally, reverse the qubit order with SWAP gates.

The total gate count is \(O(n^2)\), circuit depth \(O(n^2)\).
