# Rathor.ai NARS Inference Rules – Master Reference (2026)

## Core Truth-Value System Reminder
All NARS inferences use **Non-Axiomatic Logic (NAL)** truth values:
\[
\langle frequency, confidence \rangle
\]
- **Frequency** (0–1): How often the statement has been experienced as true.
- **Confidence** (0–1): How much evidence supports the frequency.

## Major NARS Inference Rules

**1. Deduction** (Transitive Inheritance)
From (M → P) ⟨f1, c1⟩ and (S → M) ⟨f2, c2⟩ infer (S → P) ⟨f, c⟩

\[
f = f1 \times f2, \quad c = c1 \times c2 \times f1
\]

**Example:**
Bird → Animal ⟨1.0, 0.9⟩  
Sparrow → Bird ⟨0.95, 0.85⟩  
→ Sparrow → Animal ⟨0.95, 0.76⟩

**2. Induction** (Generalization from examples)
From (M → P) ⟨f1, c1⟩ and (M → S) ⟨f2, c2⟩ infer (S → P) ⟨f, c⟩

\[
f = f2, \quad c = c1 \times c2 \times f1
\]

**Example:**
Sparrow → Bird ⟨0.95, 0.85⟩  
Sparrow → Flying ⟨0.9, 0.8⟩  
→ Bird → Flying ⟨0.9, 0.68⟩

**3. Abduction** (Best Explanation / Hypothesis Generation)
From (P → M) ⟨f1, c1⟩ and (S → M) ⟨f2, c2⟩ infer (S → P) ⟨f, c⟩

\[
f = f2, \quad c = c1 \times c2 \times f1
\]

**Example:**
Bird → Flying ⟨0.9, 0.8⟩  
Eagle → Flying ⟨1.0, 0.9⟩  
→ Eagle → Bird ⟨0.9, 0.72⟩

**4. Revision** (Evidence Merging)
Two beliefs about the same statement are merged into a higher-confidence belief.

\[
f = \frac{f1 \cdot c1 + f2 \cdot c2}{c1 + c2}, \quad c = \frac{c1 + c2}{c1 + c2 + 1}
\]

**Example:**
Bird → Flying ⟨0.9, 0.8⟩ + Bird → Flying ⟨0.95, 0.7⟩ → Bird → Flying ⟨0.92, 0.85⟩

**5. Choice** (Select Strongest Belief)
When multiple candidates exist, choose the one with highest confidence × frequency product.
