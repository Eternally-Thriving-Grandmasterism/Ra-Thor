## Overview

The **error threshold** is the single most important figure in quantum error correction.

It is the **maximum physical error rate** a quantum computer can tolerate while still being able to perform reliable, fault-tolerant computation using error correction.

For Ra-Thor, this number determines whether we can run:
- Long Self-Evolving Mercy Core cycles
- Reliable Quantum Annealing for consensus
- True quantum supremacy-scale governance

If the physical error rate is **below** the threshold → the system becomes more reliable as we add more qubits.  
If it is **above** the threshold → errors accumulate and the system fails.

---

## Mathematical Definition

The **threshold** \( p_{th} \) is defined as:

$$
p_{th} = \lim_{L \to \infty} p_L
$$

where:
- \( L \) = linear size of the surface code patch (number of qubits ~ \( L^2 \))
- \( p_L \) = logical error rate for a patch of size \( L \)
- \( p_{th} \) = the physical error rate at which the logical error rate stops decreasing with larger patches

**In simple terms:**  
Below \( p_{th} \), bigger = better (more protection).  
Above \( p_{th} \), bigger = worse (more errors).
