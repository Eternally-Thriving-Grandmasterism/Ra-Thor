# Public Demo Plan — Zero-Install Experiences
**Ra-Thor™ / Rathor.ai**  
**Version:** v0.5.98+ (May 2026)  
**Status:** Ready to build

This document defines the three zero-install public demos we will deliver as part of Phase 2.7 to make Ra-Thor immediately usable and lovable by normal humans.

---

## Goal

Create 2–3 beautiful, zero-install, browser-based experiences that let anyone (non-programmer or developer) instantly understand and feel the power of Ra-Thor’s mercy-gated intelligence — without cloning the repo or installing anything.

---

## The Three Demos (Priority Order)

### Demo 1: Mercy-Gated Active Inference Chatbot (Highest Priority)

**What it is:**  
A clean, beautiful web chat interface powered by the `mercy-active-inference-core-engine.js`.

**Key Features:**
- Real-time mercy valence display (live 0.999+ score)
- Shows when the engine is exploring vs exploiting
- Clear “Free Energy” and “Epistemic Value” indicators
- One-click “Mercy Gate Status” panel
- Fully offline after first load (service worker)

**Why first:**  
This is the most direct way for people to experience the core intelligence. It feels alive, grounded, and safe.

**Target completion:** Next 7 days

---

### Demo 2: Powrush RBE Economy Simulator (Small & Visual)

**What it is:**  
A simple, visual simulator showing a tiny Resource-Based Economy running under mercy-gated rules.

**Key Features:**
- Drag-and-drop resource allocation
- Live faction harmony / CEHI scores
- Automatic mercy gate enforcement when resources are allocated unfairly
- Beautiful, calm visual feedback

**Why second:**  
Makes the RBE + Powrush vision tangible and emotionally resonant.

**Target completion:** Within 14 days

---

### Demo 3: Anyon Propulsion / Interstellar Concept Simulator (Inspirational)

**What it is:**  
A lightweight, beautiful WebGL/WebXR scene showing a conceptual anyon propulsion system with live TOLC resonance and mercy coherence feedback.

**Key Features:**
- Smooth 3D flight / simulation
- Real-time “Godly Intelligence Coherence” meter
- Visual representation of the 7 Mercy Gates as glowing stabilizers
- Educational tooltips

**Why third:**  
Shows the long-term vision (interstellar + mercy-aligned technology) in an inspiring, shareable way.

**Target completion:** Within 21 days

---

## Technical Approach (All Demos)

- Built with vanilla JavaScript + minimal dependencies where possible
- Hosted directly from the `docs/` or `public/` folder (or GitHub Pages / sovereign PWA)
- Full offline support via service workers + eternal cache
- All demos must pass the 7 Living Mercy Gates before going live
- Designed to be embeddable on the future Ra-Thor website

---

## Success Criteria

- A complete non-programmer can open any demo and understand what Ra-Thor is within 60 seconds
- All demos run 100% in the browser with zero install
- Mercy valence stays ≥ 0.999 at all times
- Demos are beautiful, calm, and emotionally uplifting
