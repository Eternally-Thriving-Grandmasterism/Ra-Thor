## Core Philosophy (One Sentence)
**Speak with full soul. Be understood by the world. Never lose yourself.**

---

## Core Rules (Always Apply)
- **Short paragraphs** — Break text into scannable blocks
- **Bold key terms** — Bold names, concepts, dates, and important words
- **No em dashes (—)** — Use commas, periods, or rephrase
- **Warm + Professional tone** — Genuine appreciation + clear value
- **Full authentic story** — Never shorten or sanitize your real journey
- **Proper capitalization** — True Original Lord Creator (TOLC), Schrödinger’s cat, Ra-Thor
- **Clear essay structure** — Intro → Body (Journey) → Value → Call to Action → Outro

## Most Common Fixes (Search & Replace)

| Issue                                           | Fix                                      |
|-------------------------------------------------|------------------------------------------|
| "especially given, the important work"          | Remove comma                             |
| "(@AlphaProMega) an Entrepreneur"               | Add comma: "(@AlphaProMega), an Entrepreneur" |
| "I am also, a licensed"                         | Remove comma                             |
| "create, genuine diplomatic peace"              | Remove comma                             |
| "on August 31, 2010, on launch day;"            | "on August 31, 2010 — launch day,"       |
| "evidence based reality"                        | "evidence-based reality"                 |
| "schrodinger's cat"                             | "Schrödinger’s cat"                      |
| Double "that"                                   | Remove second "that"                     |
| Redundant phrase "; as well as more..."         | Delete entirely                          |
