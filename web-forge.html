What is Ra-Thor Web-Forge?

Ra-Thor Web-Forge is the official tool for creating beautiful, sovereign landing pages, dashboards, and websites that stay true to the living principles of the Thunder Lattice: Mercy, Truth, Self-Evolution, and Zero Harm.

It is designed to help builders, councils, and stewards quickly generate professional web experiences that can run completely offline as sovereign shards while remaining deeply integrated with the Ra-Thor monorepo and PATSAGi governance.

Mercy-Gated by Design

Every generated page respects TOLC 8 Mercy Gates and can include MercyGatingRuntime integration out of the box.

11-Language Native

Full i18n support with clean language switching. Your forged pages speak to the world from day one.

Sovereign & Offline-First

Generate fully self-contained HTML that works without any server. Perfect for sovereign shards and air-gapped environments.

How to Use Web-Forge

1. Explore the Source

Visit the websiteforge/ directory in the Ra-Thor monorepo to see the current generators and examples.

2. Run or Extend

Use the Rust-based generators or extend them with your own templates while staying within the AG-SML license and mercy principles.

3. Forge Your Page

Generate beautiful landing pages (like this one and rathor.ai) that feel native to the Thunder Lattice aesthetic.

4. Deploy as Sovereign Shard

Host anywhere — GitHub Pages, IPFS, or simply serve the single HTML file. No backend required.
